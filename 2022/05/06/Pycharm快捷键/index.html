Pycharm快捷键

发表于2022-05-06|更新于2022-05-06|高效工具
|阅读量:

Pycharm for Mac

⌘ + ⌥ + L 使代码遵守pep8规范（自动整齐代码）

⌘ + B 查看申明 / 查看源代码

⌘ + W 关闭当前文件标签

⌘ + F 查找

⌘ + R 替换

⌘ + K 更新到VCS

⌘ + D 复制当前一整行

⌘ + X 剪贴当前一整行

⌘ + ⇧ + V 粘贴缓存（历史粘贴板全部内容）

⌘ + O 查找类名

⌘ + ⇧ + A 查找动作

⌘ + P 参数提示

⌘ + - 折叠代码

⌘ + = 展开代码

⌃ + Tab 切换文件窗口

⌃ + R 运行

⌃ + ⌥ + R 换文件运行

⌃ + D 启动调试

⌃ + ⌥ + H 调用层次结构

⇧ + F6 重命名

⇧ + 回车 下一行

⌘ + ⇧ + ⌥ + [ 选择代码开始处] 选择代码结束处
